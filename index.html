Stalewall

A semi-practical demo for a wallpaper api

Hi!
This is a small demo for Stalewall, an api that returns a random wallpaper from various sources. You can find a hosted version of this api at stalewall.spacefall.workers.dev.

Please remember that the wallpapers returned from this api should be used for personal use only (except maybe unsplash images).

Settings Providers:
Bing Chromecast Spotlight Fire TV Earth View Unsplash APOD*
